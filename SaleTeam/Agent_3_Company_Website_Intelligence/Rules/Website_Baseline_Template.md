# Quy tắc: Mẫu baseline website

`Website baseline` là ảnh chụp trạng thái website tại một thời điểm để Agent 3 có thể:
- so sánh thay đổi giữa các lần quét
- tránh ghi `thay đổi website` khi chưa có mốc so sánh
- giữ ngữ cảnh ổn định cho account research

## Mẫu khuyến nghị

```md
## Baseline website
- Ngày tạo baseline:
- Ngày cập nhật baseline gần nhất:
- Sitemap dùng để lập baseline:
- Trạng thái baseline:

### Snapshot cốt lõi
- Headline chính:
- CTA chính:
- Mô tả công ty ngắn:
- Nhóm sản phẩm / giải pháp chính:
- Dấu hiệu ICP nổi bật:
- Dấu hiệu GTM nổi bật:
- Dấu hiệu tăng trưởng nổi bật:

### Snapshot theo nhóm trang
- Homepage:
- About:
- Product / Solution:
- Pricing:
- Customer / Case study:
- Careers:
- Blog / Newsroom:
- Contact / Locations:
```

## Trạng thái baseline nên dùng
- `Chưa có baseline`
- `Đã tạo baseline`
- `Đã cập nhật baseline`
- `Baseline hạn chế`

## Quy tắc
- Nếu đây là lần quét đầu tiên của account, Agent 3 phải tạo baseline trước khi ghi nhận thay đổi.
- Nếu chỉ quét được một phần website, được phép ghi `Baseline hạn chế`.
- Không được dùng `Website change log` như bằng chứng thay đổi nếu chưa có baseline hoặc mốc so sánh đáng tin.
