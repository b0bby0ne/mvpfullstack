# Quy tắc: Mẫu brief nghiên cứu website trực tiếp

Khi người dùng muốn Agent 3 chạy độc lập, có thể thu intake theo mẫu sau:

```md
# Website_Research_Brief

- Nguồn vào: Người dùng trực tiếp
- Tên run:
- Tên công ty / đối tượng nghiên cứu:
- Website / Domain:
- Mục tiêu nghiên cứu:
- Câu hỏi trọng tâm:
- Trang hoặc nhóm trang cần ưu tiên:
- Khung thời gian cần so sánh:
- Mức output / độ sâu mong muốn:
- Ghi chú khác:
```

## Quy tắc
- Nếu người dùng không cung cấp đủ tất cả trường, Agent 3 chỉ hỏi lại phần tối thiểu còn thiếu.
- `Mức output / độ sâu mong muốn` có thể là:
  - `Nhanh`
  - `Chuẩn`
  - `Chuyên sâu`
