# Output Guide

Mỗi nghiên cứu trend nên có một thư mục riêng trong `Output/`.

Ví dụ:
- `Output/2026-03-weekly-ai-trends/`
- `Output/social-commerce-signals-q2/`

Bên trong nên có:
- `Executive_Summary.md`
- `Trend_Report.md`
- `Product_Implications.md`
- `Marketing_Implications.md`
- `Signal_Log.md`
- `Master_Index.md`
